## How to Run

1. Clone the repository:
```

git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)

```

2. Install dependencies:
```

npm install

```

3. Start the server:
```

node app.js

```

or (if using nodemon):
```

nodemon app.js

```
